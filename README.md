# `blctl`

🏮 A library and CLI tool for controlling the monitor backlight on Linux based operating systems.

## Why?

When I started school, I decided to install Arch Linux on my laptop because I wanted to know what all the fuss was about. I quickly realized that I had to do everything myself, including managing the brightness of my laptop using a small software called [brightnessctl](https://github.com/Hummer12007/brightnessctl) made by [Hummer12007](https://github.com/Hummer12007).

After some time, I had a mental breakdown because when I changed the brightness, it happened instantly, which didn't look good to the eye at all. Also, the brightness perceived by the eye isn't proportional to the value set in the app. Instead of cloning the repository and proposing changes, being the egoist that I am, I've decided to make my own tool. Jokes aside, I think it's a great opportunity to learn a new language, so I went with Rust.
